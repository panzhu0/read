# 29 天下没有白吃的午餐

- Review

```
我小时候老爸也喜欢用讲故事给我讲道理，不过他讲的故事不多，可能不久就会重复，不过前几次听故事确实很有意思，总能吸引我的注意力，还能学到背后的道理，当我16岁以后，基本就不爱听他的故事了，因为经常重复，而且我也不想听那些故事，我觉得故事讲道理效率太慢了，直接告诉我应该怎么做就好了
```

- 信的读后感

```
每个人出生就是消费者，而且一开始总有人给他们免费提供东西，这可能会给他们带来错误的期望，因为现实世界不是为人类设计的，现实世界不会自动依照人的意愿产生产品，因此要培养自己成为生产者，给别人提供产品，这样才能获得别人给你提供的产品。不然就只能通过出卖自己的劳动力，时间来获取别人的产品
(如果把自己的时间、劳动力作为产品也行吧，不过这种产品比其他产品更珍贵，得想办法构建不需要自己花费时间就能获取利益的东西=> 资产)

我觉得现实是很现实的，有的人想富人就应该去救济他们，因为富人剥削了他们，认为富人应该为他们的贫穷负责。我觉得他们这么想很正确，很符合逻辑。而富人不愿意直接给穷人钱，或者说根本不愿意施舍任何东西给穷人，因为他们认为穷人不够努力，或者很愚昧，总是在某些重要的决定上做愚蠢的决定，我觉得也对。

那么问题在哪呢，为什么都对呢，问题应该出现在这是现实世界，而不是想象中的世界，这是我当前的想法：
1.人首先要维持自己的生存和爱好，之后才是旁边的人的
2.人首先帮助最亲近的人，然后才是其他人
3.世界不是为人设计的，人要让世界完成一件事是复杂的，这不仅要与自然打交道，还需要和人打交道

```
